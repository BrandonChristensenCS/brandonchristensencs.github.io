Dungeons & Dragons Character Sheet

The back-end for a Dungeons & Dragons 5e character sheet built in C++. It allows a user to upload their character and then make rolls using their characters statistics, or make a custom roll by specifying the number of dice and the number of sides on the die. If applicable, the user can add modifiers and/or damage types to the roll. The front-end for this project is coming soon.
